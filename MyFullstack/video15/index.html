3 types of writing css

        1)Inline CSS -inside html tag using style attribute
                      -try to avoid the use of this because 
                      it make the code more confusing while changing something.

        2)Internal css -Using Style tag in the head tag
                        -it should be used when the html file size is small.

        3)External CSS - in  an external style sheet
                        -It should be used when u are working in a serious project.
